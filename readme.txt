== Description ==

This plugin adds a review system to your Wordpress website. It lets customers place a nice review for your work.
The reviews can be called with an Shortcode of trough a widget.

== Installation ==

1. Upload `lm_reviews.zip` via the Upload link in the WordPress plugins dashboard.
2. Activate the LM_Reviews Plugin through the 'Plugins' menu in WordPress
3. Use [lm_reviews] or use a widget to show the reviews

== Frequently Asked Questions ==

- How can i add the reviews to my website?

Use [lm_reviews] or use a widget to display the reviews

- Can i accept the review first before i show it on my website?

Yes, we have created a options page, where you can choose if you want to accept it first or show it directly.
